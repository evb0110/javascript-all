001. lina lina, lina lina lina čōza? // nōza ʕa tikkōna nizbun kōza.
002. hōš nizlōli ʕa payṯa w niṯyōli, // immi p-tarba hōši minčaḳyōli. // hačči l-ḥōlax zellax w_ana l-ḥōli, // w əʕrōba čḳōmez ʕa payṯa ḳmōza.
003. p-tarba zlalla p-tarba ḥmiččil ḥōṯa, // wayba m-ḳorba naffīḳa mnə-ṣlōṯa. // ṭalpiṯ ḥalba ōmra: «bə-ḥḏawōṯa, // nmayṯōxi ḥalba w ʕezza w karrōza.»
004. lippiš aḳṣa m-ḳorḥil leppa b-zōte. // lippi minniš mō čḥammal ažōte. // ti rōḥem eḥḏa m-ġayərlə blōte, // felkil ʕomre yā ʕumri žunnōza.
005. ʕaža xann əm-riḥmūṯa čḏayyība? // ṯḳilliš baḥar ʕa blōta čġayyība. // lō nyaḏḏeʕ hašši l-ġayre čixṭība, // mnə-ʕṣofra emḥar bakkar ʕa žhōza.
006. allxiṯ ʕemma, allxaṯ ʕimmi p-šūḳa, // allxaṯ ḳummi m-zaḳzūḳa l-zaḳzūḳa. // ḥmačča emma zīla w b-īḏa ḏlūḳa, // hazzačča w leppa layyef ʕa hzōza.
007. ōmar: «emḥar ḥašoppa maščūṯa, // battaḥ naskar w ənraḳḳḏell əḥḏūṯa. // ti čū ṯēle w lā ōṯ ġappe mrūṯa, // la yimḥell ġayre payṯe mnə-ḳzōza.»
