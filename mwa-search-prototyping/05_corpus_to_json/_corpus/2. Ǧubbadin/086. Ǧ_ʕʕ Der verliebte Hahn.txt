010. amrōle: «ana nḥaḏḏīya // b-ḏahbi bi-yzayyaninnay, // ḥluffašīṯa wa_ayṯiya // ʕaya hī aḥsa minnay?

012. amella: «yā šunīṯa, // biš-šūsub ḏīča l-ḳanpōra // wa-ličin ḥluffašīṯa // sība šaḳfi ṣarṣūra.
